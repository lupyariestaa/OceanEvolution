# 🎮 Gameplay

OceanEvolution is a browser-based evolution game.

Your mission:

- 🐟 Eat smaller fish
- ⚠️ Avoid dangerous predators
- ⭐ Gain experience
- 🧬 Unlock new evolutions
- 👑 Become the king of the ocean
